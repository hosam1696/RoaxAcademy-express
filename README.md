Installation
-----
to get you a copy of the project 

first you got to clone it 

```bash
$ git clone https://github.com/hosamsam/RoaxAcademy-express.git
```

then 
```bash
$ cd RoaxAcademy-express
$ npm install
```

and run 
```bash
$ npm run dev:mon
```
 server will be listen on port `3000` by default
